- 아래의 json형태의 가게 정보를 보고 [주종]과의 페어링이 가장 좋은 가게 1~5곳 추천해.
- 기준 : 1. 메뉴 2. 분위기 3. 날씨 4. 리뷰 5. place_url 정보

- 상세 요구 사항 :
1. 실제 해당 [주종] 판매 가능성이 높은 곳만 선정.
2. 업종/이름/메뉴 만으로 판단이 모호한 경우, 반드시 place_url을 보고 판단해라.
3. place_url은 "http://place.map.kakao.com/{가게 id}" 형태
4. 억지로 추천하지말고, 추천할 곳이 없다면 빈 배열 [] 반환.
5. 뷔페, 패밀리레스토랑, 제과점 등 페어링이 부적절한 업종은 우선 제외.
6. 메뉴를 절대 지어내지 말고 구글 검색을 통한 실제 메뉴명을 넣어라.
7. 출력은 반드시 JSON 형태로 반환. (설명 및 마크다운 금지)

- r (추천 문구) 작성 규칙:
1. place_url에서 확인한 이 가게의 '시그니처 메뉴'와 '리뷰에서 극찬한 포인트'를 반드시 포함해. (업종명 언급 X)
2. [주종]과 해당 메뉴가 왜 잘 어울리는지 구체적인 이유를 한 문장 섞어줘.
3. 말투는 친한 친구에게 맛집을 영업하는 느낌으로 생생하게 작성해.
4. 이모지 1~2개를 적절히 섞고, 110자 이내로 작성해.
5. 날씨는 부정적 영향만 피하라:
   - "추운 날 → 차가운 음료" X
   - "비오는 날 → 야외" X
   - 긍정적/중립적 언급만 (실내, 따뜻한 분위기 등)
- s는 추천 점수
- 출력 형식: JSON array  (id,r,s 1-1000)
